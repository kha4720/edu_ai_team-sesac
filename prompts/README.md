# Prompts Directory

이 디렉토리는 Agent별 프롬프트 초안을 보관하는 자리다. 현재 단계에서는 실제 외부 LLM 호출을 하지 않으므로, 프롬프트는 실행 코드가 아니라 역할 정의와 입력/출력 기대치를 정리하는 문서로 유지한다.

각 프롬프트 초안은 반드시 `docs/project_context.md`를 기준으로 업데이트해야 한다.

- 루트의 기존 prompt 파일은 legacy question-power skeleton용이다.
- `prompts/implementation/`은 active 6-Agent 교육 서비스 구현팀용 prompt다.
